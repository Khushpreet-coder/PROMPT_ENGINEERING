# PROMPT_ENGINEERING DAILY DIARY
DAY-1 

    a) Introduction to Prompt Engineering and understanding how AI responds to instructions.
    b) Learned the importance of clear prompts and how structure improves output quality.
    c) Overview of types of prompts such as Instruction-Based, Role-Based, Few-Shot, Zero-Shot, etc.
    d) Practiced writing simple prompts and identifying good vs bad prompts.

DAY-2 

    a) Introduction to Generative AI and how it creates text, images, audio, and video.
    b) Explored different AI tools like ChatGPT, Gemini, Claude, LLaMA, Whisper, and DALL·E.
    c) Understood difference between LLMs, diffusion models, and multimodal models.
    d) Tried generating sample text and simple images using online tools.

DAY-3

    b) Explored how transformer architecture works using self-attention.
    c) Understood stages of model training such as pre-training, fine-tuning, and RLHF.
    d) Learned practical uses of LLMs in chatbots, translation, code, and summaries.

DAY-4 

    a) Explored ChatGPT Playground and Gemini AI Studio interfaces.
    b) Learned how temperature, max tokens, and system prompts affect responses.
    c) Practiced generating structured outputs in JSON and bullet formats.
    d) Compared responses between different model versions.

DAY-5 

    a) Built a simple webpage summarizer using Python and Gemini API.

    c) Practiced generating AI-based images using Hugging Face and Stability API.
    d) Understood base64 conversion for saving generated images.

DAY-6

    a) Explored all features of Google AI Studio including grounding and multimodal inputs.

    c) Tested Flash vs Pro models to compare speed and accuracy.
    d) Understood how to manage API keys and integrate them into Python projects.

DAY-7 

    a) Learned Speech-to-Text using Whisper model.

    d) Explored how transcription can be used in real-world applications.

DAY-8 

    a) Learned Text-to-Speech (TTS) using ElevenLabs API.
